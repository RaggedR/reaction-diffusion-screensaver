# Music-Reactive Reaction-Diffusion Screen Saver

A browser-based screen saver that grows [Gray-Scott](https://groups.csail.mit.edu/mac/projects/amorphous/GrayScott/) maze patterns and reconfigures them in response to music.

## How It Works

The screen fills with intricate maze patterns generated by a reaction-diffusion simulation running on the GPU. When music plays, the system detects the tempo via FFT bass energy analysis, locks onto the beat, and sends expanding ripple waves from the center of the screen on each beat. The ripples force the maze to reconfigure as they pass through — a wavefront of destruction followed by regrowth.

Colours drift slowly through greens, teals, blues, and purples.

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:5173` and click to start. Grant microphone access when prompted — the screen saver listens to music through your mic.

## No Microphone?

Without a mic, the screen saver runs in autonomous mode — maze patterns grow and drift, but without beat-triggered ripples.

## Tech Stack

- **Vite** — build tool
- **Three.js** — WebGL rendering (ping-pong framebuffers)
- **GLSL** — Gray-Scott simulation + HSL colour mapping
- **Web Audio API** — FFT-based bass energy extraction and tempo detection

## Based On

The Gray-Scott simulation shaders are adapted from [reaction-diffusion-playground](https://github.com/jasonwebb/reaction-diffusion-playground) by Jason Webb.
